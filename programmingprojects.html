Prominent Colors Kotlin Designed, programmed, and published an Android app that can take an image, from photo gallery or the user's camera, and find the top 5 most used colors by pixel in the image. Programmed an algorithm that uses a HashMap to keep track of the colors and its frequency, along with a sorting algorithm that places the colors into Pairs on a LinkedList. Published on the Play Store and added interstitial advertisements when the user wants to scan another image.
Pathfinding and NavMesh Study Java Programmed a Java 2D AI agent to traverse a NavMesh in the shortest Path using the A* algorithm. Programmed an algorithm to automatically develop a triangulated NavMesh based on the avaiable walls.

DFA of Decimal Floating Point Literals Java Recreated Java's Decimal Floating Point literal using a Deterministic Finite Automaton state diagram. Translated the DFA diagram into a program to practice practical uses of state machines.
PDA of Decimal Floating Point Expressions Java An extension of the DFA of Decimal Floating Point Literals project. Crafted a Pushdown Automata state diagram and translated it into a program in order to simulate Java decimal floating point expressions.
Matrix Multiplication Study Java Created 3 algorithms of multiplying matrices together (Brute Force, naive divide and conquer algorithm, and Strassen's algorithm) in order to find and compare the time complexities of each algorithm as the matrix size grows.
